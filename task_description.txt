Создание маршрутов для разных страниц
Используя файлы в архиве rounting.zip, а конкретно в файле index.php создать маршруты для разных страниц и подключить соответствующие файлы шаблонов через require_once.

Необходимо создать маршруты:
/ - index.html
/blog - blog-full.html
/services - services.html
/team - team.html
/contact-us - contact.html

P.S.: Разрешается реализовать на основе аннотаций к методам контроллеров как мы делали на уроке
