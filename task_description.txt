Работа с автозагрузчиком классов по стандарту PSR-4

1. Создать себе проект с работающим отладчиком
2. Исследовать принцип работы готового класса автозагрузки https://www.php-fig.org/psr/psr-4/examples/#class-example с помощью отладчика.
    2.1 Создать каталог src
    2.2 Создать любой класс в каталоге/подкаталоге src
    2.3 По примеру урока создать файл автозагрузки vendor/autoload.php и поместить в него класс из ссылки
    2.4 Повторить проделанное на уроке
    2.5 Понять предназначение функции spl_autoload_register
3.Создать свой собственный класс автозагрузки (упрощённый) по примеру класса Psr4AutoloaderClass
